# Hardware photos

Drop photos of the physical HexBee build in this folder, then they'll be
embedded in [docs/HARDWARE.md](../../docs/HARDWARE.md) and
[JOURNAL.md](../../JOURNAL.md).

**Use JPG or PNG** so GitHub renders them inline. iPhone `.heic` files won't
display on GitHub — convert them first (or drop them in and they can be
converted to JPG).

Suggested filenames (helps captions stay meaningful):

| File | Shows |
|------|-------|
| `scout.jpg` | The ESP32-S3 Scout board |
| `scout-case.jpg` | Scout in its enclosure |
| `hive.jpg` | Raspberry Pi 3B+ Hive + USB SSD |
| `kit-full.jpg` | The whole kit together |
| `dashboard.jpg` | Live capture on the Hive dashboard |
| `field-iphone.jpg` | iPhone field companion in use |
