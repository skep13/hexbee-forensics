🐝 Forager — live host

Read-only agent. Run it on a target computer to capture processes, network, logons, autoruns, USB history and recent files into the evidence chain.

USB stick (no install): plug in the HexBee Forager stick and double-click RUN-WINDOWS.bat → choose Collect. Or from a shell:

hexbee-forager --hive {{ base_url }} --key {{ ingest_key or 'YOUR-INGEST-KEY' }} collect

Continuous monitoring: replace collect with watch --interval 60.

🔬 Comb — disk & media

Triage a mounted image or drive on the Queen: hashing, carving, partitions, EXIF/GPS, browser history — findings flow straight into this Hive.

Point-and-click: run the Comb UI, paste the path, click Scan.

hexbee-comb serve

Or scan and upload directly:

hexbee-comb scan /mnt/evi --hive {{ base_url }} --key {{ ingest_key or 'YOUR-INGEST-KEY' }}

🐝 Scout — USB hardware agent

The ESP32-S3 field device reports USB/host activity over Wi-Fi. Firmware is in hardware bring-up; rehearse the full pipeline with the simulator:

python scout_sim.py --rest {{ base_url }} --key {{ ingest_key or 'YOUR-INGEST-KEY' }} --scenario incident
